## QUICK REFERENCE CARDS

### The Three Pillars (1-minute)

```
1. COACHING PLATFORM
   - Accessible to beginners (templates, AI suggestions)
   - Powerful for experts (Lab mode, sliders, simulations)
   - Reference: MacroFactor, Stronger by Science

2. CLIENT APP
   - Goal: 5-min daily entry (beautiful, zero friction)
   - Data types: Performance, biometrics, nutrition, wellness
   - Success: Client WANTS to log daily

3. ECOSYSTEM (Future, Foundations Now)
   - 2026: Wearables
   - 2027: Smart restaurants
   - 2028+: Sport centers
   - Moat: Closed loop = network effects
```

### Anti-Frustration (1-minute)

```
✅ Single-tap data entry
✅ Smart defaults (remember last session)
✅ Real-time feedback (< 300ms)
✅ Visual over verbal (charts, not prose)
✅ Coach empowerment (suggests, doesn't replace)

❌ Multi-step forms
❌ Guilt-driven notifications
❌ Forcing changes without coach approval
❌ Data silos
❌ Dark patterns (leaderboards, streak fear)
```

### Decision Checklist (2-minute)

```
1. Helps adherence? (client sticks longer?)
2. Removes friction? (less clicks/confusion?)
3. Feeds flywheel? (data → intel → personalize → data?)

If all YES → investigate full decision matrix
If any NO → might not be core (could be Tier 2/3)
```

### 18-Month Roadmap (1-minute)

```
Q2 2026: MVP (program builder, client app, nutrition)
Q3 2026: Tier 2 (wearables, export, analytics, AI)
Q4 2026: Ecosystem start (restaurant MVP, 2-3 partners)
2027: Scale (50+ restaurants, 20+ gyms, 2k coaches)
2028+: Moat formation (network effects, data flywheel)
```
